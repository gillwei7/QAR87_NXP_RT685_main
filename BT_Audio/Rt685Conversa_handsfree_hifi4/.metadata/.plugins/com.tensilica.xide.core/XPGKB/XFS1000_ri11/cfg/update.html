RI-2021.8 Xplorer Update Information

Xtensa Xplorer update 3020 is available, and will be offered for download and installation. The current installed version can be checked in the Help -> About dialog.

The following updates to Xtensa Xplorer are included with this patch:

IDNatureDiscovered InSummary
60981 Bug RI-2021.8Xplorer C editor incorrectly greyed out the active part of code under #ifdef when the macro is defined in active target. This has been fixed.
26976 Improvement RG-2015.0Changes to the dynamic variable assist dialog so that the variables are sorted alphabetically.
59460 New Feature RI-2021.8Support has been added to the Xplorer debugger for watchpoints on eTIE Wires.
60336 Bug RI-2021.7Xplorer hardware profile module stalled when compiled with xt-clang. This has been fixed and support for '-hwpg' has been added.
60788 Improvement RI-2021.8Xplorer support for XNNC 2.4.1 and also improvements to preferences to more reliably detect XNNC version.
61056 New Feature RI-2021.8Xplorer support for 'single cycle step' while debugging a program. To enable this feature, Xplorer debug view adds 'Cycle Stepping Mode' toolbar button.
61098 Bug RI-2021.8Xplorer pre build step did not resolve the 'xt_tools' dynamic variable correctly when the active config switched between different Xtensa releases. This has been fixed.
61343 Bug RI-2021.8On Linux platform, XOCD manager could not launch and connect to OpenOCD. This has been fixed.
61388 Bug RI-2021.6The Xplorer auto patch mechanism would automatically attempt to install patches from the XPG for shared Xplorer installations and installations without write permissions. This has been fixed and would be effective for any subsequent patch.
59586 Improvement RG-2016.4Xplorer improvement for 'On-chip Debug' launch configuration to support passing arguments to the hardware target.
60603 Bug RI-2021.8Xplorer multi-core debug module did not create debug console for cores other than core0. This has been fixed.
60739 Bug RI-2021.7Xplorer C editor reported 'Syntax error' warnings (without any details) when the active set contained RI-2021.7/RI-2021.8 tools. This has been fixed.
60849 Improvement RI-2018.0An Xplorer 'Debugger' preference has been added to allow configuration of max lines per message in GDB trace consoles.
62042 Bug RI-2021.8During software upgrades of FuSa builds to non-FuSa releases, the build wizard would not refresh the build error messages and re-calculate the minimal and optimal hardware version from the build. This has been fixed.
